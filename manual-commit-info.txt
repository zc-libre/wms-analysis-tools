Git Commit Template

Commit types:
1. feat: new feature
2. fix: bug fix
3. docs: documentation changes
4. style: formatting, missing semi colons, etc
5. refactor: code restructuring
6. test: adding tests
7. chore: maintenance tasks

Example commit message structure:
----------------------------------------
type: brief description

Detailed explanation of what changes were made,
why they were necessary, and any important implementation details.

-Agent Generated Commit Message
----------------------------------------

Instructions for committing:
1. Select a commit type from the list above
2. Write a brief description (max 72 chars)
3. Write a detailed body paragraph explaining the changes
4. Install Git if not already installed: https://git-scm.com/downloads
5. Run the following commands in terminal:
   - git add .
   - git commit -m "Your complete commit message from above"
   - git push
